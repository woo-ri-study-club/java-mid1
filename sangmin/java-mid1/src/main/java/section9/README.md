# section 9 미션
내부 클래스, 정적 중첩 클래스, 지역 클래스, 익명 클래스, 람다를 활용하여 레스토랑 예약 관리 시스템을 구현한다.

## 필수 기능
- 테이블 관리 및 예약 기록
- 예약 정보를 추가/삭제 및 테이블 상태 업데이트.
- 특정 조건에 따라 예약 또는 테이블을 검색.

## 클래스 구조
Restaurant
└── Table (정적 중첩 클래스)
├── Reservation (내부 클래스)
└── TableStatusUpdater (지역 클래스)

익명 클래스와 람다는 main 메서드 안에 구현을 하여 활용한다.

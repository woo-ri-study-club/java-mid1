# 우리의 스터디 클럽 - 자바 중급1 출사표
## 개요
올해 마지막달인 12월, 우리는 자바 중급1 스터디를 시작합니다. 이 스터디는 자바 중급1 강의를 듣고, 강의 내용을 정리하고, 실습을 통해 학습한 내용을 복습하고자 하는 목적으로 진행됩니다.

## 목표
이번 중급 강의에서는 Object 클래스, 불변 객체, String 클래스, 래퍼 클래스, 열거형(ENUM), 날짜와 시간, 중첨클래스, 내부 클래스, 예외처리 등 자바에서 중요한 개념들을 배웁니다.
중요한만큼 개념을 한번에 습득하기 어렵고 시간도 오래걸릴 것으로 예상됩니다. 이번 스터디 부터 일정이 러프해진만큼 강의 이외에도 찾아보면서 모자른 지식을 채워나가는 방향으로 진행하려고 합니다.

## 마무리
자바 고급편까지 마무리하는날까지 열심히 달려봅시다.
